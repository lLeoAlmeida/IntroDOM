Introdução ao DOM (Document Object Model)

Uma breve introdução sobre o DOM e suas aplicações

Posted by Prof. Leonardo em 24 de Agosto 2023

Introdução

O Document Object Model, ou DOM, é uma parte fundamental do desenvolvimento web moderno e desempenha um papel vital na interação entre os navegadores e o conteúdo das páginas da internet. Para entender o DOM, imagine a web como um vasto repositório de documentos e informações. Cada página da web é um documento, e o DOM age como uma representação estruturada desse documento, permitindo que os programadores acessem, manipulem e modifiquem o conteúdo e o estilo das páginas web de maneira dinâmica.

Ao carregar uma página web, o seu navegador analisa todo o código HTML e monta uma estrutura internamente chamada de árvore DOM, que é uma representação a estrutura do documento HTML. Nessa estrutura, cada elemento, comentário ou texto do documento HTML é representado como um objeto, denominado nó. A estrutura DOM é utilizada para manipular o documento HTML dinamicamente, utilizando programação, com a DOM API e a JavaScript.

O DOM é uma interface de programação de aplicativos (API) que permite aos desenvolvedores web interagir com os elementos individuais de uma página da web, como parágrafos, imagens, formulários e muito mais. Com o DOM, nós iremos ver como aprender as seguintes interações:

  1. Seleção de Elementos
  2. Manipulação de Conteúdo
  3. Atributos e Estilos
  4. Eventos

Seleção de Elementos

A seleção de elementos é o processo de obter referências para os elementos lá no HTML usando JavaScript. Podemos usar diversos métodos para selecionar elementos, como "getElementById", "querySelector", "getElementsByClassName", "getElementsByTagName", entre outros.

  • getElementById

    Este método é usado para selecionar um elemento com base em seu atributo id.

    No exemplo acima, ao apertar o botão 'Teste Aqui', ele irá inserir a mensagem "Titulo", na região h1 pelo seu ID, onde acessamos essa região através do comando getElementById.

  • querySelector

    O querySelector é um método poderoso que permite selecionar elementos usando seletores CSS. Ele retorna o primeiro elemento que corresponder ao seletor especificado.

    No exemplo abaixo, o código HTML possue dois elementos h1 Meu Título1 e Meu Título2 respectivamente, após o uso do querySelector ele irá modificar apenas o Meu Título1.

  • getElementsByClassName

    O getElementsByClassName é usado para selecionar todos os elementos que têm uma determinada classe. Ele retorna uma coleção (array-like) de elementos correspondentes.

    No exemplo a seguir, ele irá apresentar pelo alert, todo conteudo que pertence a classe "paragrafo"

  • getElementsByTagName

    O getElementsByTagName é usado para selecionar todos os elementos que têm uma determinada tag (como < p > , <h1> , etc.). Ele também retorna uma coleção de elementos correspondentes.

    O exemplo abaixo, é semelhante a do getElementsByClassName, contudo ele está pegando o conteudo pertecente a todos os elementos <p>.

Manipulação de Conteúdo

A manipulação de conteúdo com JavaScript envolve a seleção de elementos HTML existentes em uma página da web e a alteração de suas propriedades, conteúdo e estilo. Iremos utilizar duas das maneiras mais comuns de modificar o conteúdo de elementos HTML são usando as propriedades textContent e innerHTML. Vamos entender como cada uma delas funciona e ver exemplos de uso.

  • textContent

    A propriedade textContent permite acessar ou modificar o texto dentro de um elemento HTML. Ela retorna o texto contido no elemento, incluindo todo o texto dentro de seus elementos filhos, mas não interpreta o HTML como código. Isso significa que as tags HTML dentro do texto são tratadas como texto simples.

    Exemplo de Leitura usando textContent: 

                            
// Suponha que tenhamos o seguinte HTML:
// <p id="meuParagrafo"> Este é um 
//<strong > parágrafo </strong> de exemplo.</p>
                            
// Acessando o conteúdo de texto do elemento 
    
const paragrafo = document.getElementById("meuParagrafo");
const texto = paragrafo.textContent;
console.log(texto); 
// Saída: "Este é um parágrafo de exemplo."

    Exemplo de Escrita usando textContent:

                            
// Modificando o conteúdo de texto do elemento <p>
paragrafo.textContent = "Texto alterado dinamicamente.";
  • innerHTML

    A propriedade innerHTML permite acessar ou modificar o conteúdo HTML dentro de um elemento. Diferentemente de textContent, innerHTML interpreta o HTML como código e pode ser usado para adicionar ou modificar elementos HTML dentro de um elemento pai.

    Exemplo de Leitura usando innerHTML: 

                       
// Suponha que tenhamos o seguinte HTML:
// <div id="minhaDiv"><p>Este é um 
//<strong>parágrafo</strong> de exemplo.</p></div>

// Acessando o conteúdo HTML do elemento <div>
const minhaDiv = document.getElementById("minhaDiv");
const conteudoHTML = minhaDiv.innerHTML;
console.log(conteudoHTML);
// Saída: "<p>Este é um 
//<strong>parágrafo</strong> de exemplo.</p>"
    Exemplo de Escrita usando innerHTML: 
                        
// Modificando o conteúdo HTML do elemento 
    
minhaDiv.innerHTML = "<p>Conteúdo alterado dinamicamente..</p>";

    • Atributos e Estilos

    Os atributos e estilos são propriedades associadas aos elementos HTML que podem ser acessadas e manipuladas por meio de JavaScript. Eles desempenham papéis diferentes, mas ambos são fundamentais para a manipulação dinâmica de elementos em uma página da web.

    Atributos
    Os atributos são informações adicionais associadas a um elemento HTML. Cada elemento HTML pode ter atributos que definem várias características, como identificação, classe, fonte da imagem, URL do link, valor de entrada, entre outros. Os atributos são definidos diretamente no código HTML e podem ser acessados e modificados por meio do JavaScript. Alguns exemplos comuns de atributos incluem, id, class, src, href, value.

    Exemplo de acesso e modificação de atributos: 

                    
// Acessando o atributo 'src' de uma imagem
var imagem = document.getElementById('minhaImagem');
var source = imagem.getAttribute('src');

// Modificando o atributo 'href' de um link
var link = document.getElementById('meuLink');
link.setAttribute('href', 'https://google.com');

    Estilos
    Os estilos se referem às propriedades de estilo CSS de um elemento HTML. Eles controlam a aparência e o layout do elemento, como cor, tamanho da fonte, margens, preenchimento, etc. Assim como os atributos, os estilos também podem ser acessados e modificados por meio do JavaScript. Para acessar os estilos de um elemento, você pode usar a propriedade style. Alguns exemplos de propriedades de estilo incluem, color, font-size, margin, padding.

    Exemplo de acesso e modificação de estilos: 

                    
// Acessando o estilo 'color' de um elemento de parágrafo
var paragrafo = document.getElementById('meuParagrafo');
var corTexto = paragrafo.style.color;
                    
// Modificando o estilo 'background-color' de um elemento de div
var div = document.getElementById('minhaDiv');
div.style.backgroundColor = 'lightblue';

    Eventos

    Os eventos do DOM são ações ou ocorrências que acontecem dentro de um documento HTML, geralmente causadas por interações do usuário ou por outras atividades no navegador. Esses eventos permitem que você detecte e responda a ações, como cliques do mouse, pressionamentos de teclas, carregamento de páginas, movimento do mouse, entre outros.

    addEventListener

    O addEventListener é um método em JavaScript que permite que você adicione um ouvinte de eventos a um elemento HTML. Isso significa que você pode instruir o navegador a ouvir eventos específicos (como clique de mouse, pressionamento de tecla, carregamento de página, etc.) em um elemento específico do DOM e executar uma função quando esse evento ocorrer.

    Exemplo de sintaxe do addEventListener: 
                    
elemento.addEventListener(evento, function);

    • elemento: O elemento HTML ao qual você deseja adicionar o ouvinte de eventos.
    • evento: O nome do evento que você deseja ouvir (por exemplo, "click", "keyup", "DOMContentLoaded", etc.).
    • function: A função que deve ser executada quando o evento ocorrer.
