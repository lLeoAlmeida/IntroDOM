Introdução ao DOM

"Ok Tio Léo, mas o que é esse tal de DOM? Praque ele serve?"

- Aluninhos do 1ºINFO


Então Jovens Padawans, o DOM (Document Object Model) é uma interface de programação fornecida pelos navegadores que representa a estrutura da página HTML e permite que os desenvolvedores acessem e manipulem dinamicamente os elementos dessa página. Ou seja, com o DOM podemos utilizar a linguagem de Programação JavaScript para selecionar elementos da página, alterar seus conteúdos, estilos e atributos, criar novos elementos, remover elementos existentes e muito mais.

Estrutura do DOM

Ao carregar uma página web, o seu navegador analisa todo o código HTML e monta uma estrutura internamente chamada de árvore DOM, que é uma representação a estrutura do documento HTML. Nessa estrutura, cada elemento, comentário ou texto do documento HTML é representado como um objeto, denominado nó. A estrutura DOM é utilizada para manipular o documento HTML dinamicamente, utilizando programação, com a DOM API e a JavaScript.

Por Exemplo, como construiriamos uma Arvore DOM do código a seguir? (Árvore será construida no quadro durante a aula). 

          
            <!DOCTYPE html>
            <html>
            <head>
              <title>Titulo da Pagina</title>
            </head>

            <body>
              <h1>Primeira Pagina</h1>
              <p>Hello World!<p>
            </body>
            </html>

Seleção de Elementos

A seleção de elementos é o processo de obter referências para os elementos lá no HTML usando JavaScript. Podemos usar diversos métodos para selecionar elementos, como "getElementById", "querySelector", "getElementsByClassName", "getElementsByTagName" , entre outros.

  • getElementById

    • Este método é usado para selecionar um elemento com base em seu atributo id. Ele retorna o elemento com o ID especificado. 

                        <!DOCTYPE html>
                        <html>
                        <head>
                          <title>Exemplo de getElementById</title>
                        </head>
                        <body>
                          <h1 id="titulo">Título da Página</h1>
                          <button onclick="exemplo1()">Teste Aqui  </button >
                        </body>
                        </html>
                      
                    
                    
                  /*script.js*/
                  // Selecionar o elemento com ID "titulo"
                  var tituloElemento = document.getElementById('titulo');
                   function exemplo1(){
                  alert(tituloElemento.textContent); // Exibirá o texto dentro do elemento h1
                   }


  • querySelector

    • O querySelector é um método poderoso que permite selecionar elementos usando seletores CSS. Ele retorna o primeiro elemento que corresponder ao seletor especificado. 

                
                <!DOCTYPE html>
                <html>
                      <head>
                        <title>Exemplo de querySelector</title>
                      </head>

                      <body>
                        <h1 id="destaque">Título da Página</h1>
                      <button onclick="exemplo2()">Teste Aqui  </button >
                      </body>
                </html>
            
          
                    
                  /*script.js*/
                  // Selecionar o elemento com a classe "destaque"
                  var destaqueElemento = document.querySelector('.destaque');
                   function exemplo2(){
                  console.log(destaqueElemento.textContent); // Exibirá o texto dentro do elemento h1
                   }


  • getElementsByClassName

    • O getElementsByClassName é usado para selecionar todos os elementos que têm uma determinada classe. Ele retorna uma coleção (array-like) de elementos correspondentes. 

                 
                 <!DOCTYPE html>
                 <html>
                       <head>
                         <title>Exemplo de getElementsByClassName</title>
                       </head>
 
                       <body>
                         <p class="paragrafo">Parágrafo 1</p>
                         <p class="paragrafo">Parágrafo 2</p>
                         <p class="paragrafo">Parágrafo 3</p>

                         <button onclick="exemplo3()">Teste Aqui  </button >
                         
                       </body>
                 </html>
             
           
                     
                  /*script.js*/
                  // Selecionar todos os elementos com a classe "paragrafo"
                  var paragrafos = document.getElementsByClassName('paragrafo');
                   function exemplo3(){
                  // Iterar sobre a coleção para exibir o texto de cada parágrafo
                  for (var i = 0; i < paragrafos.length; i++) {
                      console.log(paragrafos[i].textContent);
                     }
                 }


  • getElementsByTagName

    • O getElementsByTagName é usado para selecionar todos os elementos que têm uma determinada tag (como <p>, <h1>, etc.). Ele também retorna uma coleção de elementos correspondentes. 

                 
                 <!DOCTYPE html>
                 <html>
                       <head>
                         <title>Exemplo de getElementsByClassName</title>
                       </head>
 
                       <body>
                         <p>Parágrafo 1</p>
                         <p>Parágrafo 2</p>
                         <p>Parágrafo 3</p>

                         <button onclick="exemplo4()">Teste Aqui  </button >
                         
                       </body>
                 </html>
             
           
                     
                  /*script.js*/
                  // Selecionar todos os elementos <p>
                    var paragrafos = document.getElementsByTagName('p');
                     function exemplo4(){
                    // Iterar sobre a coleção para exibir o texto de cada parágrafo
                    for (var i = 0; i < paragrafos.length; i++) {
                        console.log(paragrafos[i].textContent);
                     }
                    }
